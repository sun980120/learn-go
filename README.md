# GO-LANG-GIN-PROJECT
GO + GIN + GORM 공부 프로젝트


## 🖥️ 프로젝트 소개
GO의 기초부터 응용하여 공부한 내용을 적용하는 프로젝트입니다.
<br>

## 🕰️ 개발 기간
* 24.09.23일 - Current

### 🧑‍🤝‍🧑 맴버구성
- 개발자 : 홍우성 - 전체 프로젝트 관리 및 통합 & 형상관리


### ⚙️ 개발 환경
- `Go 1.23.0`
- `JDK 1.8.0`
- **IDE** : GoLand(2024.2)
- **Framework** : Gin(1.10.*)
- **Database** : MariaDB(11.*)
- **ORM** : GORM
